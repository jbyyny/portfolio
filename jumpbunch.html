Project Description

JumpBunch is a site targeted for parents and guardians of young children. It's an extracirricular year-round fitness program for ages 2-12. This Silicon Valley franchise site was designed and built as the new design template for all U.S. Jumpbunch franchises.

Site features included large imagery of kids having fun displayed in an immersive parallax web experience. Coaches profiles along with over 75 different kinds of exercises are featured. Additional features include a join form, class schedules and locations, registration and online payment.

Early designs, flows and maps.
